# MySTM32 开发仓库

欢迎来到我的 STM32 开发仓库！

这是一个开源项目，旨在分享我在 STM32 微控制器开发中的经验和代码，希望它能为你的开发提供一些帮助。

## 项目结构

- **DFU-test/**: 包含 DFU 测试相关的代码和配置。
- **FreeRTOS_LED/**: 使用 FreeRTOS 实现 LED 控制的示例。
- **Keypad_Polling/**: 按键轮询的实现代码。
- **test/**: 测试项目和相关文件。

## 特性

- 提供多种 STM32 项目的示例代码。
- 包含常用的驱动程序和中间件。
- 支持 FreeRTOS 和 USB 功能。
